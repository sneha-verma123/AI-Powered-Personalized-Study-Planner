# AI-Powered-Personalized-Study-Planner
📚 AI-Powered Personalized Study Planner

An AI-based application that creates personalized study plans for students based on their goals, subjects, and available time. This project uses Generative AI to help users study more efficiently with smart and adaptive schedules.

🚀 Features
✨ Generates personalized study schedules using AI
📅 Accepts user input like subjects, goals, and study hours
🧠 Uses NLP models for intelligent planning
🔄 Flexible and adaptive study plans
🌐 Interactive UI using Gradio
🛠️ Tech Stack
Python
Gradio – for building the user interface
Transformers (Hugging Face) – for AI/LLM functionality
pip – for package management
📂 Project Structure
├── app.py              # Main application file (core logic)
├── requirements.txt    # Project dependencies
├── README.md           # Project documentation
└── other files...
app.py → Handles user input, AI processing, and output generation.
⚙️ Installation & Setup

Clone the repository:

git clone https://github.com/sneha-verma123/AI-Powered-Personalized-Study-Planner.git

Navigate into the project directory:

cd AI-Powered-Personalized-Study-Planner

Install required libraries:

pip install -r requirements.txt
▶️ Run the Application
python app.py

After running, a Gradio interface will open in your browser.

🧪 How It Works
User enters:
Subjects or topics
Study goals
Available time
The application processes the input using a Transformer-based model.
AI generates a personalized study plan.
The plan is displayed through the Gradio interface.
